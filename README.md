# Cadastro Joystick Journal

Projeto desenvolvido para praticar formulários HTML através da criação de uma página de cadastro temática voltada ao universo gamer.

## 🎮 Sobre o projeto

O site simula o cadastro de usuários para a plataforma fictícia **Joystick Journal**, uma comunidade focada em notícias, análises e conteúdos relacionados ao mundo dos jogos.

O objetivo foi criar um formulário organizado, funcional e com identidade visual temática.

## 📚 Conteúdos praticados

* Formulários HTML
* Inputs
* Labels
* Fieldset e Legend
* Select
* Textarea
* Checkbox
* Radio Button
* Upload de arquivos
* Validação básica com `required`
* Links
* Estruturação de conteúdo

## 🛠️ Tecnologias utilizadas

* HTML5

## 📋 Funcionalidades do formulário

### 👤 Dados pessoais

* Nome completo
* E-mail
* Data de nascimento
* Telefone
* Nacionalidade
* Cidade
* Senha

### 🎯 Preferências

* Gêneros favoritos
* Plataforma de preferência

### 🖼️ Perfil

* Upload de foto
* Biografia do usuário

### 📜 Termos de uso

* Aceite obrigatório para envio do formulário

## 🚀 Objetivo

Este projeto foi criado com foco em:

* praticar construção de formulários
* entender diferentes tipos de inputs
* organizar informações de usuários
* melhorar estruturação HTML
